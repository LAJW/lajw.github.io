Wrona History Menu

The ultimate history management extension for Chromium browsers.

Chrome Web Store Opera addons Fork on Github

Features

  • View and restore recently closed tabs and windows
  • Browse through recent history
  • View tabs from other devices
  • Keyboard navigation
  • History entry removal
  • Resize popup
  • Synchronize settings with other devices (using native Chrome sync)
  • Lightning-fast (takes only 90ms to load on an atom netbook)
  • Customize icon color
